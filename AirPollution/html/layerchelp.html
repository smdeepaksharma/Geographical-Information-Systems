LAYER CONTROL

Layers added to the map can be reordered using the Layer Control tool

To change the layer order :

Step 1: Select a layer in the toc



Step 2: Click on Layer Control



Step 3: Select one of the following options

  • Promote layer :

    Moves the selected layer up by one level

  • Demote layer :

    Moves the selected layer down by one level

  • Promote to top :

    Moves the selected layer to the top of the toc


NOTE : A layer 'must' be selected in order to enable the layer control options
