LEGEND EDITOR

To open legend editor :
Step 1: Go to File menu
Step 2: Click on Legend Editor



Properties of a layer can be edited using the following options

  • Symbols
  • Labels
  • General

SYMBOLS

You can select one of the following methods to redraw the features

  • One Symbol
  • Graduated Symbols
  • Unique Symbols


COLOR

The color of the symbols can be changed using this option

STYLE

The marker symbol can be changed using Style menu

LABELS

The symbols can be labeled based on various attributes of the
features. Additionally, user can change the font, color, size and
placement of the labels
